# About the covid traker

Developed to track the current covid stuations around the world,

## Technologies used

1. React
2. Redux
3. Sass,
4. Open API
5. Git
6. React Hooks

### To run the project

`clone repo` <br/>

`npm install | yarn install` <br/>

`npm start | yarn start` <br/>

`open browser and check localhost: 3000` <br/>
